Gym Tonic

Privacy Policy

Last updated: 2026-06-29

Gym Tonic is a workout app for creating and following gym workout cards. The app is designed to work without an account, backend, or cloud sync in V1.

Data Stored On Device

Gym Tonic stores workout data locally on the user's device, including:

  • workout cards and exercises;
  • sets, repetitions, weight, rest times, and notes;
  • workout session history;
  • rest feedback settings;
  • manual backup files exported by the user.

This data stays on the device unless the user manually exports a backup file and chooses where to store or share it.

Accounts And Cloud Sync

Gym Tonic does not require an account.

Gym Tonic does not provide cloud backup, multi-device sync, or server-side storage in the V1 release.

Data Collection

Gym Tonic does not collect personal data, health data, analytics events, advertising identifiers, or usage tracking data in the V1 release.

Notifications

Gym Tonic can use local notifications to alert the user when a workout rest timer finishes. These notifications are generated on the device.

Notification settings can be changed inside the app settings or through the operating system settings.

Manual Backups

Gym Tonic can export a local JSON backup file. The user controls where this file is saved or shared.

Because backup files may contain workout history and notes, users should store them only in places they trust.

Third Parties

Gym Tonic does not send workout data to third-party services in the V1 release.

The app uses platform APIs and local device storage to provide its core functionality.

Children's Privacy

Gym Tonic is not designed for children and does not knowingly collect data from children.

Changes

This policy may be updated if Gym Tonic adds features such as ads, purchases, analytics, accounts, cloud sync, or online backup.

Any future feature that changes data handling must be reflected in this policy before release.
